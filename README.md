#### Instalação:

```shell
$ yarn
```

---

#### Rodando a aplicação:

```shell
$ sudo docker-compose up
```

_OBS_ : Para gerar a documentação das rotas da API, instale o apidoc globalmente com:

```shell
$ yarn global add apidoc
```

Então, execute o seguinte comando

```shell
$ yarn doc
```
